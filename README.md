# ai-ml-module-ece-lab
AI/ML module for E2 285 at IISc

This is a 3-week module covering the basics of Python programming and machine learning. 

Each week's material is packaged in a Jupyter notebook with instructional notes embedded inside. Additional programming exercises are included at the end of each notebook. 

Week 1: Python primer - data structures, program flow, functions, object-oriented programming

Week 2: Linear prediction, linear regression, optimization - stochastic gradient descent, automatic differentiation 

Week 3: Handwritten digit image classification based on the MNIST dataset using PyTorch, convolutional neural networks
